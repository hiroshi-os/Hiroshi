The `hiroshi doctor` command runs a battery of health checks:

```bash
hiroshi doctor
```

- Ollama connectivity and model availability
- `~/.hiroshi` workspace permissions
- SQLite memory store integrity
- `config.toml` validation
- Active agent definitions in `AGENTS.md`
