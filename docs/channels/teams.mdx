---
title: Microsoft Teams Integration
description: Dispatch markdown payloads and Adaptive Cards to Microsoft Teams channels.
---

Hiroshi supports automated outbound messaging to MS Teams workflows by remapping markdown payloads into Teams-compliant Adaptive Card structures.

## ⚙️ Configuration Setup

Configure MS Teams Webhook URLs in your `~/.hiroshi/config.toml`:

```toml
[teams]
enabled = true
workflow_url = "https://prod-XX.westus.logic.azure.com:443/workflows/..."
allowed_channels = ["General"]
```

## 📋 Adaptive Cards Remapping

Teams does not support standard markdown table formatting. When forwarding logs or search results to Microsoft Teams, the Hiroshi core engine automatically serializes text payloads into a sequence of Adaptive Card text blocks and key-value column layouts.
